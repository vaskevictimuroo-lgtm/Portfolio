Resend bot 



Бот для модерации пользовательских предложений. Пересылает сообщения от пользователей в чат модераторов и обратно.

Функции:

Принимает сообщения от пользователей (текст, фото, файлы)
Пересылает в чат модераторов с информацией об отправителе
Модераторы отвечают через reply
Ответы автоматически отправляются пользователям

Технологии:

Aiogram 3.x
SQLite
Асинхронная архитектура

Установка
git clone <репозиторий>
cd bot
python -m venv venv
venv\\Scripts\\activate
pip install -r requirements.txt

Настройка:

Создайте .env файл:

Получите токен у @BotFather
Добавьте бота в чат модераторов и пропишите команду /chatid

BOT\_TOKEN=ваш\_токен\_от\_BotFather
MODERATOR\_CHAT\_ID=id\_чата\_модераторов



Запуск:
python main.py

Использование:
Пользователи: пишут боту → сообщения пересылаются модераторам
Модераторы: отвечают reply в чате → ответ идет пользователю
